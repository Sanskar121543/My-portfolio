Research & Projects

Some of my recent work and personal projects

Defect Analysis in Machine Components (ISRO)

CNN • ResNet50 • PyTorch • Python

A collaborative research project with ISRO to detect and classify manufacturing defects in aerospace components for predictive maintenance.

  • Architected an end-to-end CNN pipeline using ResNet50 transfer learning.
  • Built a ~10k-image augmentation suite and applied class-balanced sampling.
  • Achieved ~94% accuracy with 40% fewer false positives vs the baseline model.
  • Delivered a deployable inspection prototype adopted for pilot testing.

Portfolio Optimization Web App

Flask • Markowitz MPT • Cvxpy • Python

A transparent, scenario-driven risk-return allocation tool for retail investors and allocators.

  • Implemented Markowitz Modern Portfolio Theory with rolling-window covariance.
  • Achieved an average Sharpe ratio of ≈0.72 vs 0.45 baseline in historical backtests.
  • Used convex optimisation (cvxpy) for efficient asset allocation.
  • Visualizer supports over 100+ scenario comparisons for robust analysis.

Interest-Rate Modeling Web App

Monte-Carlo • Vasicek Model • Plotly • Python

A web application to robustly simulate and price interest-rate dependent instruments under stochastic-rate dynamics.

  • Implemented the Vasicek model with 10,000 Monte-Carlo simulation paths.
  • Developed parameter estimation routines from historical data.
  • Reduced mean pricing error by ~0.35% compared to a naive baseline.
  • Enabled interactive scenario analysis across 100+ parameter sets using Plotly.
